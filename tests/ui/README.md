# UI Login Tests (Playwright + JavaScript)

## Описание проекта

Небольшой учебно-практический проект по автоматизации UI-тестирования страницы логина с использованием **Playwright** и **JavaScript**.

Цель проекта - продемонстрировать базовые навыки QA Automation:

- написание e2e UI-тестов
- работа с негативными сценариями
- проверки бизнес-логики
- визуальная регрессия интерфейса

Тесты написаны для демо-приложения:  
https://demoqa.com/login

---

## Что покрыто тестами

### UI-тесты

- Проверка отображения формы логина
- Негативный сценарий входа:
  - ввод некорректного логина и пароля
  - проверка сообщения об ошибке
  - проверка, что пользователь остаётся на странице логина
- Визуальная регрессия формы логина при ошибке авторизации

---

## Используемый стек

- **JavaScript (ES6)**
- **Node.js**
- **Playwright**
- Playwright Test Runner
- Snapshot testing (visual regression)

---

## Визуальная регрессия

В проекте используется snapshot-тестирование Playwright:

создаётся эталонный скриншот формы логина

при последующих запусках выполняется сравнение

позволяет отлавливать нежелательные изменения UI
