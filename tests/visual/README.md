# Visual Regression Test: Register Buttons

Этот тест демонстрирует проверку визуальной регрессии веб-интерфейса с использованием Playwright на JavaScript.

Тест проверяет, что кнопки регистрации на странице не смещаются по верстке и остаются в заданном расположении.

## Что делает тест

1. Переходит на страницу регистрации:  
   `https://demoqa.com/register`

2. Скроллит контейнер кнопок `.buttonWrap` в видимую область.

3. Проверяет визуальное состояние контейнера кнопок с эталонным скриншотом:

   ```js
   await expect(buttonsContainer).toHaveScreenshot(
     "screenshots/register-buttons.png"
   );
   ```

4. Если текущее состояние отличается от эталонного, тест падает и сохраняет скриншот для диагностики.

⚠️ В текущей версии теста верстка искусственно ломается для демонстрации работы теста.

## Результат

Тест падает, если кнопки смещены или изменен внешний вид.

Скриншоты сохраняются в папке screenshots/ для наглядной диагностики.

## Цель

Демонстрация навыка QA Automation и визуальной регрессии.

Пример для портфолио начинающего QA Automation инженера с использованием Playwright и JavaScrip
