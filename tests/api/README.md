# API Login Test (Playwright)

Этот тест демонстрирует проверку API веб-приложения с использованием Playwright на JavaScript.

Тест проверяет **позитивный сценарий логина** на демо-сайте `https://demoqa.com`.

## Файл теста

`tests/api/login-api.spec.js`

## Что делает тест

1. Отправляет POST-запрос на эндпоинт логина:  
   `https://demoqa.com/Account/v1/Login`

2. Проверяет, что сервер возвращает **статус 200**.

3. Проверяет, что в теле ответа есть свойство `token` (успешная авторизация).

4. Логирует JSON ответа в консоль для наглядности.
