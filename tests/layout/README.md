# Layout Regression Test — Register Page Buttons

## Описание

Автоматизированный UI-тест, написанный на **Playwright + JavaScript**,  
предназначен для проверки корректного выравнивания кнопок **Register** и **Go To Login** на странице регистрации.

Тест демонстрирует:

- проверку layout-логики UI
- работу с координатами элементов (`boundingBox`)
- фиксацию реального UI-дефекта
- визуальное подтверждение бага (screenshot / snapshot)

Тест написан для демо-приложения:  
https://demoqa.com/register

---

## Что проверяет тест

- кнопки **Register** и **Go To Login** отображаются на странице
- кнопки должны находиться на одной горизонтальной линии (одинаковая ось Y)
- визуальная регрессия блока с кнопками

---

## Известный дефект (Known Issue)

На текущей версии демо-сайта кнопки **Register** и **Go To Login**
смещены по вертикали примерно на **17px** и не выровнены по оси Y.

### Поведение теста

- тест **намеренно падает**, так как фиксирует существующий UI-дефект
- дефект подтверждён:
  - координатами элементов (`boundingBox`)
  - визуальным скриншотом блока с кнопками

Тест помечен как **known issue** и может быть активирован повторно после исправления верстки.
