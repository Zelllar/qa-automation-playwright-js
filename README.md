Учебные автотесты

Добро пожаловать в репозиторий с учебными автотестами, созданными в рамках самостоятельного изучения QA и автоматизации тестирования.

Содержание репозитория

В репозитории собраны автотесты, разработанные на JavaScript для проверки веб-приложений:

API-тесты – проверка статус-кодов, структуры ответов, корректности работы API.

UI-тесты – проверки пользовательского интерфейса, включая расположение элементов на странице, сравнение визуальных элементов по изображению.

Валидация ввода - тесты на корректность обработки данных пользователем, например, проверка ввода пароля.

Используемые технологии

Язык программирования: JavaScript

Инструменты: DevTools, Node.js

Фреймворки: учебные тестовые скрипты без сторонних тестовых фреймворков (или укажи Playwright / Jest, если использовал)

Цель репозитория

Репозиторий демонстрирует базовые навыки автоматизации тестирования, понимание теории QA и практическое применение тест-кейсов в учебных проектах.
Все тесты были созданы с целью практики и закрепления знаний по тестированию веб-приложений.
